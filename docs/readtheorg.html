綠色資訊區塊 Tip

#+begin_tip
If you have tables (for example) that you want to ignore during export, one possibility
is to use *comment blocks* or ~:noexport:~ subtrees.  Another possibility is to
*use* non-exported *drawers* (see #+OPTIONS: d:).
#+end_tip

If you have tables (for example) that you want to ignore during export, one possibility is to use comment blocks or :noexport: subtrees. Another possibility is to use non-exported drawers (see #+OPTIONS: d:).

自訂巨集 Macro (記得空行換行)

#+MACRO: color @@html:<span style="color: $1">$2</span>@@

{{{color(blue, This text is colored in blue.)}}}

{{{color(red, This other text is in red.)}}}

This text is colored in blue.

This other text is in red.
